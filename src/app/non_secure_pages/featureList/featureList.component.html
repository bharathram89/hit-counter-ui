Feature List

This is an on going project and will be releaseed in multiple phases so not all features are avalaible but do check back or subscribe to our Release Updates.

Phase 1

Players will be able to register to our application and be able to create basic profile with social media connection to share your stats.

Fields will be able to register to our application and be able to create basic profile with social media connection to share your stats.

Phase 2

Players will be able to add detailed information about their loadouts (camo,gun,attachments...). They will be able to select the loadout they are running for a game.

Fields will be able to add detailed information about their fields like maps, hours. They will be able to setup pre registration so players can register to play prior to the day of.

Phase 1

Players can buy a device which they can use on the field to track stats then they will be able to see their previous games and track stats on a cell phone application.

Fields will be able to see the number of registered players at the games that day, they will be able to track the number of games played that day in the field and # of total kills and other stats of the day from the field.

Phase 1

Players will be able to create custom game types and play the games type of the fields and default games types given.

Fields will be able create a custom game type for their field and the players who select that field will be see those game types to play.

Phase 2

Players device will connect to a device on field for more custom games like objective based

Fields will be able to setup and play with game control devices.
